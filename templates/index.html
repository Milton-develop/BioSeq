To view the 3D structure, we first need to identify the protein's PDB Code.

Identify on BLAST

Instructions:
1. Make sure to tick that option suggests "Show results in a new window"
2. Click the Blue button to run a BLAST search.
3. Look at the search results and find the PDB ID from the table under the accession column (e.g., 4HHB).
4. Open the 3D Protein Viewer from the main portal sidebar on the dashboard and type that ID.
